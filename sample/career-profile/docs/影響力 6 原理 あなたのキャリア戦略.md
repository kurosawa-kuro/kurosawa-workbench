### 具体的アクションプラン（90 日ロードマップ）

1. **Week 1–2｜返報性を仕込む**

   * Qiita: *「Vertex AI Pipelines × BigQuery MLOps ハンズオン」* を公開
   * GitHub: bq-first リポジトリの README に PoC スライドサンプルへのリンクを追加 → Fork した人があなたを参照しやすい環境を整備

2. **Week 3–4｜一貫性を可視化**

   * README の冒頭に ✔ To-Do リスト（GCP Professional ML Engineer 取得予定・PoC 公開予定）を追加
   * X（Twitter）固定ツイートで「リーダー・コンサルタントへのシフト宣言」を投稿

3. **Week 5–6｜社会的証明を収集**

   * 元クライアント・上司 2 名に LinkedIn 推薦依頼（GCP 基盤整備・リーダーシップの観点）
   * Qiita の人気記事に「Vertex Vector Search 活用事例」を追加 → 新規 LGTM を獲得

4. **Week 7–8｜好意ネットワークを拡張**

   * GCP ユーザーコミュニティ / Slack で Vertex AI の悩みを共有・回答 → 類似性で信頼獲得
   * 勉強会後に PoC スライド + サンプルコードを共有（Notion ページ or GitHub） → 感謝で好意ループ

5. **Week 9–10｜権威づけ**

   * オンラインカンファレンス CFP を提出（テーマ: *”Vertex AI Pipelines × BigQuery で作る再現可能な MLOps 基盤”*）

6. **Week 11–12｜希少性マーケ**

   * Wantedly プロフに “GCP AI 基盤コンサルタント / チームリーダー — 新規案件募集枠：月 100h まで” と記載
