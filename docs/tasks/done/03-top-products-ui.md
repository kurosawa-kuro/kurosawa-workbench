# トップ画面・商品一覧画面 UI

## Goal

`/`（トップ）と `/products`（商品一覧）を営業デモで見せられる見た目に仕上げる。

## Value

デモの第一印象を決める画面。「AI付き EC だ」と一瞬で伝わるかどうかがここで決まる。

## Scope

**トップ `/`**
- ヒーローコピー「何を選べばいいか分からない」を AI が解決します
- コンシェルジュ入力欄（チップ付き）→ `/concierge` へ遷移
- CTA ボタン（AIに相談する / 商品を見る）

**商品一覧 `/products`**
- カテゴリ・価格帯・タグ絞り込み
- キーワード検索（通常検索）
- 商品カード一覧（画像プレースホルダー可）
- 「AI でおすすめ順に並べ替え」ボタン（クリックは後のタスクで実装）

## Non-scope

- AI リランキングの実際の呼び出し（ボタン表示だけ）
- 商品詳細画面の中身

## Done

- トップのコンシェルジュ入力欄から `/concierge` に遷移できる
- 商品一覧にカテゴリ絞り込みが動く
- 商品カードが全 36 件表示される（絞り込み前）

## Evidence

目視確認
